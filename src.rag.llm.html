get_answers_openai(prompt, model, max_tokens, temperature)[source]¶

Sends a request to the OpenAI API with given parameters and retrieves the response.

Parameters:

  • prompt – The prompt text to send to the OpenAI model.

  • model – The identifier of the OpenAI model to use for the completion request.

  • max_tokens – The maximum number of tokens to generate in the response.

  • temperature – The temperature setting to control the randomness of the response.

Returns:

The response from the OpenAI API.
